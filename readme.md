# 易經卜卦學習 App ☯️

這是一個專為學習與推演《易經》設計的漸進式網路應用程式（PWA）。
透過數位化的方式，重現傳統金錢卦與大衍之數（蓍草卦）的機率模型，並內建完整的 64 卦與 384 爻辭資料庫。輔以宋代大儒朱熹的解卦法則，以及「互、錯、綜」的多維度深度分析，幫助使用者更直覺地理解易經的變化之理與哲學思維。

## ✨ 核心特色

* **雙重起卦演算法**：
  * **金錢卦**：模擬三枚硬幣的對稱機率（機率分布為：老陰 1/8、少陽 3/8、少陰 3/8、老陽 1/8）。
  * **大衍之數（蓍草卦）**：重現古法非對稱機率（機率分布為：老陰 1/16、少陽 5/16、少陰 7/16、老陽 3/16）。
* **動態卦象視覺化與重點標示**：
  * 自動由下而上繪製本卦與之卦圖形，並以紅色精準標示「動爻」與「靜爻」。
  * 具備高對比度的「👉 主要參考」文字區塊，在茫茫字海中精準引導閱讀焦點。
* **正統解卦邏輯（朱熹法則）**：根據 0 到 6 個變爻的不同狀況，自動判斷並顯示最適合的解卦文本（包含乾坤二卦的「用九」、「用六」特例），並根據重心自動調整版面配置。
* **支援深淺色模式 (Dark Mode)**：全自動適應手機系統的深色主題，提供高對比、不傷眼的尊爵黑金配色與清晰的純白古文體驗。
* **支援離線操作 (PWA)**：支援安裝至手機桌面，無須網路連線亦可隨時隨地開啟使用，享受全螢幕的沉浸體驗。

## 🔍 深度分析：多維度視角

除了基本的「本卦（現況）」與「之卦（趨勢）」，本系統內建易理深度推演演算法，自動計算並展示三種延伸視角，作為進階思考的引導：
1. **互卦（看內部）**：擷取本卦內聚力，剖析事件內部正在醞釀的過程與隱憂。
2. **錯卦（看極端）**：將本卦陰陽完全反轉，探討若採取極端相反策略時的局面。
3. **綜卦（看客觀）**：將本卦上下顛倒，訓練換位思考，從對手或第三人立場看待全局。

## 📂 檔案結構

本專案採用純前端技術（Vanilla HTML/CSS/JS）開發，不依賴任何外部框架，兼顧效能與學習易讀性：

* `index.html`：應用程式主體骨架與介面排版。
* `app.js`：核心演算法（亂數生成、卦象繪製、解卦邏輯判斷、多維度卦象推演）。
* `data.js`：易經 64 卦完整資料庫（包含卦名、二進位碼、卦辭、爻辭）。
* `manifest.json`：PWA 應用程式設定檔。
* `sw.js`：Service Worker 腳本，負責資源快取與版本自動更新機制。
* `icon-512.png`：應用程式專屬太極圖示。

## 📱 安裝與使用方式

**網頁版體驗：**
直接點擊專案的 GitHub Pages 連結即可在瀏覽器中操作。

**安裝至手機桌面（iOS / Android）：**
1. 使用手機瀏覽器（Safari 或 Chrome）開啟本專案網址。
2. 點擊瀏覽器底部的「分享」按鈕（iOS）或右上角的「選單」按鈕（Android）。
3. 選擇 **「加入主畫面 (Add to Home Screen)」**。
4. 即可在桌面生成專屬 App 圖示，未來點擊即可離線使用。
