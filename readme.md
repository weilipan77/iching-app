# 易經卜卦學習 App ☯️

這是一個專為學習與推演《易經》設計的漸進式網路應用程式（PWA）。
透過數位化的方式，重現傳統金錢卦與大衍之數（蓍草卦）的機率模型，並內建完整的 64 卦與 384 爻辭資料庫，輔以宋代大儒朱熹的解卦法則，幫助使用者更直覺地理解易經的變化之理。

## ✨ 核心特色

* **雙重起卦演算法**：
  * **金錢卦**：模擬三枚硬幣的對稱機率（機率分布為：老陰 1/8、少陽 3/8、少陰 3/8、老陽 1/8）。
  * **大衍之數（蓍草卦）**：重現古法非對稱機率（機率分布為：老陰 1/16、少陽 5/16、少陰 7/16、老陽 3/16）。
* **動態卦象視覺化**：自動由下而上繪製本卦與之卦圖形，並以紅色精準標示「動爻」與「靜爻」，視覺對照一目了然。
* **正統解卦邏輯**：內建「朱熹解卦法則」，根據 0 到 6 個變爻的不同狀況，自動判斷並顯示最適合的解卦文本區塊（包含乾坤二卦的「用九」、「用六」特例）。
* **支援離線操作 (PWA)**：支援安裝至手機桌面，無須網路連線亦可隨時隨地開啟使用，享受全螢幕的沉浸體驗。

## 📂 檔案結構

本專案採用純前端技術（Vanilla HTML/CSS/JS）開發，不依賴任何外部框架：

* `index.html`：應用程式主體骨架與介面排版。
* `app.js`：核心演算法（亂數生成、卦象繪製、解卦邏輯判斷）。
* `data.js`：易經 64 卦完整資料庫（包含卦名、二進位碼、卦辭、爻辭）。
* `manifest.json`：PWA 應用程式設定檔。
* `sw.js`：Service Worker 腳本，負責資源快取與版本更新機制。
* `icon-512.png`：應用程式專屬太極圖示。

## 📱 安裝與使用方式

**網頁版體驗：**
直接點擊專案的 GitHub Pages 連結即可在瀏覽器中操作。

**安裝至手機桌面（iOS / Android）：**
1. 使用手機瀏覽器（Safari 或 Chrome）開啟本專案網址。
2. 點擊瀏覽器底部的「分享」按鈕（iOS）或右上角的「選單」按鈕（Android）。
3. 選擇 **「加入主畫面 (Add to Home Screen)」**。
4. 即可在桌面生成專屬 App 圖示，未來點擊即可離線使用。
